проект тестировался на локальном сервере, для этого использовалось приложение ampps, всю папку с проектом нужно поместить в папку "www" на локальном сервере


webapp.sql - бэкап базы данных, СУБД mysql. 
Для её восстановления нужно:
1) Перейти в управление СУБД по адресу http://localhost/phpmyadmin/ 
2) Ввести пароль (для ampps пароль mysql) 
3) Создать базуданных, желательно с названием "webapp" 
4) Перейти во вкладку импорт и поместить файл webapp.sql в соответсвующее поле и нажать кнопку импорт

Далее для открытия приложения в строке браузера перейти по адресу http://localhost/proj/
